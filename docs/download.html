Verification & Installation Steps

Step 1: Verify the Checksum (Optional but Recommended)

Verifying the checksum ensures the file hasn't been corrupted or tampered with during download.

  1. Open Command Prompt (search for "cmd" in the Start menu)
  2. Navigate to the folder containing the downloaded file using the cd command
  3. Run the following command: 
    certutil -hashfile TBICE.zip MD5
  4. Verify that the output matches: ccdc23ee47128e79965fe6488335f0e2
  1. Open Terminal
  2. Navigate to the folder containing the downloaded file using the cd command
  3. Run the following command: 
    md5 TBICE.zip
    Or on Linux: 
    md5sum TBICE.zip
  4. Verify that the output matches: ccdc23ee47128e79965fe6488335f0e2

Step 2: Extract the ZIP File

  1. Right-click on TBICE.zip
  2. Select "Extract All..."
  3. Choose a destination folder or keep the default
  4. Click "Extract"
  1. Double-click the TBICE.zip file
  2. The Archive Utility will automatically extract the contents
  3. Alternatively, in Terminal: 
    unzip TBICE.zip -d ~/Desktop/TBICE

Step 3: Run the Application

  1. Navigate to the extracted folder
  2. Double-click on TBICE.exe
  3. If Windows SmartScreen appears, click "More info" and then "Run anyway"
  4. The application will start

Note: If you get a missing DLL error, make sure you have the latest Visual C++ Redistributable installed from the Microsoft website.
